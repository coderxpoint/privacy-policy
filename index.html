Privacy Policy

Last updated: January 2026

SEO AnalyzerX is committed to protecting user privacy and maintaining transparency about how data is handled within the extension.

Overview

SEO AnalyzerX is a Chrome extension designed to analyze on-page SEO elements of the website currently selected by the user. The extension operates strictly under user control and follows Chrome Web Store privacy and security guidelines.

Information Collection

SEO AnalyzerX does not collect, store, or transmit any personal data or personally identifiable information. The extension does not collect:

  • Names, email addresses, or contact details
  • Browsing history or website usage patterns
  • Login credentials or sensitive personal information

How the Extension Works

When a user initiates an analysis, the extension temporarily accesses the content of the active browser tab to extract publicly visible SEO-related elements such as:

  • Page title and meta description
  • Heading structure (H1–H6)
  • Text content and visible links

All analysis is performed locally within the user's browser session. No webpage data is transmitted to external servers.

Data Storage

The extension may store non-personal preferences and temporary SEO analysis data locally using Chrome’s storage API to improve user experience. This data remains on the user’s device and can be cleared at any time.

Third-Party Sharing

SEO AnalyzerX does not share data with third parties, advertisers, analytics platforms, or external services.

User Control

Users are always in control of when the extension runs. The extension does not execute automatically, does not run in the background, and only analyzes websites when explicitly requested by the user.

Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be reflected on this page with an updated revision date.
